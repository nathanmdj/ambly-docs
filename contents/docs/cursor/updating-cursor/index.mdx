---
title: Updating Cursor
description: Updating Cursor.
---

#### Step 1 Download the latest version on https://www.cursor.so/

#### Step 2 Make it executable

```bash
chmod +x cursor-0.42.4x86_64.AppImage
```

#### Step 3 Delete the old app image on /opt folder

```bash
sudo rm -rf <path-to-file>
```

#### Step 4 Move the app image to the /opt folder

sudo mv cursor-0.42.4x86_64.AppImage /opt/cursor.appimage
